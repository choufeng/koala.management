## 使用

### 常规

```bash
$ git clone https://github.com/choufeng/koala.management.git --depth=1
$ cd koala.management
$ npm install
$ npm start         # visit http://localhost:8000
```

### 在Docker中使用

```bash
// dev 
$ npm run docker:dev

// build 
$ npm run docker:build


// production dev 
$ npm run docker-prod:dev

// production build 
$ npm run docker-prod:build
```
